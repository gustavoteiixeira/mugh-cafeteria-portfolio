Coffee Shop Landing Page

-Projeto de portfólio que demonstra habilidades em HTML, CSS e JavaScript, com foco em design, responsividade e interatividade.
---------------------------------------------------------------------------------------------------------------------------------
Objetivo:

-Criar um site de cafeteria moderno.
-Destacar produtos (cafés e acompanhamentos) com cards interativos.
-Demonstrar layout responsivo para desktop, tablet e mobile.
-Praticar HTML intermediario, CSS avançado e JS básico.
---------------------------------------------------------------------------------------------------------------------------------
Tecnologias:

-HTML5
-CSS3 (Flexbox, Grid, animações, transições)
-JavaScript (menu mobile, contadores animados)
---------------------------------------------------------------------------------------------------------------------------------
Design & Responsividade:

-Layout compatível com desktop, tablets e smartphones.
-Uso de Flexbox e transições suaves.
-Botões e cards com efeitos de hover.
---------------------------------------------------------------------------------------------------------------------------------
